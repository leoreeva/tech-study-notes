# Snowflake editions
[(Pricing)](https://www.snowflake.com/en/pricing-options/)
[(Docs)](https://docs.snowflake.com/en/user-guide/intro-editions)


4 major editions:
- **Standard**: an entry-level, introductory offering providing access to core functionality (around $2.00 / per credit)
- **Enterprise**: all standard features + addresses the needs of high-growth, large-scale customers (around $3.00 / per credit)
- **Business Critical**: all enterprise features + offers functionality for enhanced security, disaster recovery, and higher levels of data protections to support the needs of organizations with extremely sensitive data, like PHI data that must comply with HIPAA and HITRUST CSF regulations (around $4.00 / per credit)
- **Virtual Private Snowflake (VPS)**: all business critical features + a completely separate Snowflake environment, isolated from all other Snowflake accounts
